@size 宽高一起写

background: #60569e; :doodle { @grid: 5; @size: 8em; grid-gap: 1px; }

@size 宽高分开写

background: #60569e; :doodle { @grid: 5; @size: 8em 2em; grid-gap: 1px; }

@max-size 宽高分开写

设置size大于max-size，则max-size起作用 background: #60569e; :doodle { @grid: 5; @size: 19em; @max-size: 8em; grid-gap: 1px; }

@min-size 宽高分开写

设置size小于min-size，则min-size起作用 background: #60569e; :doodle { @grid: 5; @size: 1em; @min-size: 8em; grid-gap: 1px; }

@size如果给单个单元格设置，就是设置单元格的高度

:doodle { @grid: 5; @size: 10em; grid-gap: 1px; } @size: 5px; background: #60569e;
