@grid属性 写法与优先级测试

Properties-@grid的写法优先级高于Attributes-grid的写法。

所以下边的代码分成了6列而不是5列 :doodle { grid-gap: 1px; @grid: 6 x 6; @size: 8em; } @row(1) { :after { content: @index; color: #fff; } } background: #60569e;

@grid与@size合并简写：@grid: colAndRow / widthAndHeight + 单位;

background: #60569e; :doodle { grid-gap: 1px; @grid: 3 / 8em; } @nth(n) { :after { content: @index; color: #fff; } }

完整写法：@grid: col x row / width height

background: #60569e; @grid: 5 x 4 / 8em 12em; :doodle { grid-gap: 1px; } @nth(n) { :after { content: @index; color: #fff; } }
