一维网格，grid行或列的数量取值范围变大

【@size: 8em;】 相当于 【width: 8em; height: 8em;】 :doodle { grid-row-gap: 1px; @size: 8em; } background: #60569e; width: @rand(5%, 100%); /* from 5% to 100% by random */ 仅当网格为一维时，行或列的最大限制为1024。 :doodle { grid-column-gap: 1px; @size: 18em; } background: #60569e; height: @rand(5%, 100%); /* from 5% to 100% by random */

使用Properties里的`@grid`属性来设置grid的值：

5 / 8em = 5x5的行数和列数。总体宽高是8em*8em :doodle { @grid: 5 / 8em; } background: #60569e; transform: scale(@rand(.2, .9));
