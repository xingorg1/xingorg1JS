@random 按0.5的比率随机指定单元格

刷新页面，变色的单元格会变化 :doodle { width: 8em; height: 8em; grid-gap: 2px; overflow: hidden; } @nth(n) { :after{ content: @index; color: #fff; } } background: #60569e; @random { background: hsl(285deg 100% 80%); }

@random(radio) 指定一个比率

指定0

:doodle { width: 8em; height: 8em; overflow: hidden; grid-gap: 2px; } background: #60569e; @random(0) { background: hsl(285deg 100% 80%); }

指定1

:doodle { width: 8em; height: 8em; overflow: hidden; grid-gap: 2px; } background: #60569e; @random(1) { background: hsl(285deg 100% 80%); }

指定0~1的一个数

:doodle { width: 8em; height: 8em; overflow: hidden; grid-gap: 2px; } background: #60569e; @random(.46) { background: hsl(285deg 100% 80%); }

@random([radio-start, radio-end]) 指定一个比率区间

@random 多次使用

:doodle { width: 8em; height: 8em; overflow: hidden; grid-gap: 2px; } @random(.2) { border-top: 1px solid #60569e; } @random(.42) { border-bottom: 1px solid #60569e; } @random(.6) { border-left: 1px solid #60569e; } @random(.82) { :after { content: ''; background: hsl(@rand(360), 60%, 70%); @size: @rand(3px); } }
