@row(n,...) 选择指定行单元格

:doodle { grid-gap: 1px; width: 10em; height: 10em; } background: #60569e; @row(2) { :after { content: @index; color: #fff; } } @row(3, 5) { background: hsl(58, 100%, 80%); :after { content: '-'; color: #60569e; } }

@row(even)/@row(odd) 偶数行或者奇数行

:doodle { grid-gap: 1px; width: 10em; height: 10em; } background: #60569e; @row(even) { :after { content: @index; color: white; } } @row(odd) { background: #9887ff; }

@col(n,...) 选择指定列单元格

:doodle { grid-gap: 1px; width: 10em; height: 10em; } background: #60569e; @col(2) { :after { content: @index; color: #fff; } } @col(3, 5) { background: hsl(58, 100%, 80%); :after { content: '-'; color: #60569e; } }

@col(even)/@col(odd) 偶数列或者奇数列

:doodle { grid-gap: 1px; width: 10em; height: 10em; } background: #60569e; @col(even) { :after { content: @index; color: white; } } @col(odd) { background: #9887ff; }

如果row和col并用，指定到同一个单元格了，算谁的？

很简单，谁在下边算谁的 :doodle { grid-gap: 1px; width: 10em; height: 10em; } @col(odd) { background: #9887ff; :after { content: @index; color: yellow; } } @row(even) { background: #60569e; :after { content: @index; color: white; } }
