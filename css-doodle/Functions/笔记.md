
# Functions
大多数方法的括号，都是可带可不带。如果没有参数的话，基本都可以不写括号（包括别名的写法上）

## @index，@index()，@i
返回当前单元格的索引值。

@index: 如果没有参数的时候，括号是可以忽略的。

别名：@i

## @row，@y

返回当前单元格所在的行数

别名：@y

## @col，@x
返回当前单元格所在的列数

别名：@x

## @size-row，@Y
返回整个grid网格的最大行数的值

## @size-col，@X
返回整个grid网格的最大列数的值

比如：grid="4 x 2";得到一个4列、2行的grid。
则 size-row得到4；size-col得到2
