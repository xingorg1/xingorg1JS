@size() 常用用法

返回单元格的个数 :doodle { grid-gap: 1px; @grid: 2 / 5em; } @nth(n) { :after { content: @size; color: #fff; } } background: #60569e;

配合@x、@y、@I一起使用，利用calc生成动态颜色值

:doodle { grid-gap: 1px; @grid: 5 / 10em; } @nth(n) { background: rgba(142, 170, 255, calc(@x * @y / @I)); :after { content: @i; color: #fff; font-size: 10px; } } :doodle { grid-gap: 1px; @grid: 5 / 10em; background: #60569e; @shape: circle; } @nth(n) { background: rgba(255, 225, 93, calc(@x * @y / @I)); @shape: star; :after { color: #fff; font-size: 10px; } }
