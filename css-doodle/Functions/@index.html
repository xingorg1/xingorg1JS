@index() 常用用法

搭配cell选择器的after伪类选择器，给cells们设置编号。 :doodle { grid-gap: 1px; @grid: 2 / 5em; } @nth(n) { :after { content: @index(); color: #fff; } } background: #60569e;

@index 不带参数时可以不用括号

:doodle { grid-gap: 1px; @grid: 2 / 5em; } @nth(n) { :after { content: @index; color: #fff; } } background: #60569e;

@i 别名的使用

@grid: 5 x 1 / 8em; @nth(n) { @place-cell: center; @size: calc(100% / 5 * @i); opacity: calc(1.1 - 1 / 5 * @i); z-index: calc(5 - @i); :after { position: absolute; top: 0; right: 0; content: @i; color: #fff; } } background: #60569e; :doodle { @grid: 5 x 1 / 8em; } @nth(n) { @size: calc(100% / 5 * @i); opacity: calc(1.1 - 1 / 5 * @i); z-index: calc(5 - @i); :after { content: @i; color: yellow; opacity: 1; } } background: #60569e;
